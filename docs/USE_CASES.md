# Personal Health Coach – Use Cases

## 1. Overview

This document outlines simple, easy-to-understand **user flows** for the Personal Health Coach AI agent. It focuses on **how users interact with the system** and the agent's responses.

---

## 2. Use Case 1: User Feeling Unwell

**Goal:** Provide support and guidance without diagnosing.

**Flow:**

1. User opens the agent and describes symptoms.
2. Agent asks follow-up questions in a warm, non-judgmental way.
3. Agent checks existing medical history or triggers Medical History Maker.
4. Safety & Ethics Guardrails ensure no diagnosis is given.
5. Agent provides precautions, lifestyle advice, and recommends seeing a doctor if needed.

**Outcome:** User receives empathetic guidance and a clear next step.

---

## 3. Use Case 2: User Seeking Wellness Improvement

**Goal:** Help a healthy user improve lifestyle habits.

**Flow:**

1. User opens the agent and shares current habits/goals.
2. Agent evaluates lifestyle data (sleep, diet, activity, stress).
3. Wellness Knowledge Layer provides best practices.
4. Agent generates actionable steps for improvement.
5. Encourages consistent, preventive wellness habits.

**Outcome:** User gets personalized lifestyle guidance and motivation.

---

## 4. Summary

These two primary use cases cover most user interactions:

* **Feeling unwell** → Safety-first guidance
* **Healthy and improving** → Lifestyle optimization

The flows are **simple, clear, and human-friendly**, showing how the agent interacts in different scenarios.
